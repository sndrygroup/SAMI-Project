# SAMI - Marketing Assistant Platform

SAMI is a marketing assistant platform that guides businesses through their entire marketing workflow, starting from the moment a customer signs up.

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Backend:** Node.js, NestJS, PostgreSQL, Prisma
- **Integrations:** Snapchat, TikTok, Meta (Facebook/Instagram), X (Twitter)
- **CRM:** Twilio WhatsApp API
- **Analytics:** Chart.js/Recharts, Mapbox/Leaflet

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
